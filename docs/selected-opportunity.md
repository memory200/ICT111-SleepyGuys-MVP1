# Lost & Found System

## Target User

University students and staff members.

## Problem Statement

Students frequently lose personal belongings on campus. Currently, there is no centralized platform to report or search for lost items, making recovery difficult.

## Current Alternative

Students usually ask friends, contact lecturers, or post on social media groups to search for lost items.

## Proposed IT Solution

The system will allow users to report lost and found items through a website. Users can search for items, view details, and contact the person who found the item.

## MVP Type

Simple Web Application using HTML, CSS, and JavaScript.
