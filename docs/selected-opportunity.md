# Selected Semester Project Opportunity

## Selected Project Title
Campus Lost and Found Reporting System

## Target User
University students, staff, and campus security personnel who need a centralized way to report or search for lost personal belongings on campus.

## Problem Statement
Students frequently lose personal items (ID cards, keys, water bottles, electronics) on campus and lack a single, reliable platform to report or check for found items. Currently, information is scattered across informal chat groups or buried in physical security offices.

## Current Alternative
* Asking in large messaging groups (Telegram, Viber, Messenger).
* Physically walking to campus security offices or reception desks.
* Posting on informal social media pages.

## Proposed IT Solution
A simple web-based Lost and Found portal where students can submit reports of lost or found items with details/photos, and search an organized list of reported items.

## Why This Opportunity Was Selected
Selected based on NUF scoring (New, Useful, Feasible):
* **New:** Replaces chaotic chat messages with a structured, searchable database.
* **Useful:** Directly addresses a recurring pain point for campus students.
* **Feasible:** Can be prototyped within a semester using web technologies without hardware requirements.

## Why This Is Feasible Without IoT or Advanced Cybersecurity
This project is purely software-based. It relies on standard web forms and database records (e.g., HTML/CSS/JS with Airtable or Google Sheets backend) to store and query text/image data, eliminating the need for IoT hardware sensors or complex security deployments.

## Possible MVP Type
Simple web app prototype (HTML/CSS/JS connected to a database API) or Figma clickable prototype.

## Initial Validation Plan for Lab 03
* **Who to ask:** 10–15 university students and campus security staff.
* **What evidence to collect:** Qualitative notes on how often they lose items, how they currently search, and satisfaction with existing channels.
* **Question to answer first:** How many students currently fail to recover lost items using chat groups or physical inquiries?
