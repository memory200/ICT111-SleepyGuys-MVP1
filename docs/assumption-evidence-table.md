# Assumption vs. Evidence Table

| Team Assumption | What Users Actually Said / Did (Evidence) | Validation Status |
| :--- | :--- | :---: |
| **Assumption 1:** Students don't report items because they are lazy. | **Evidence:** Students want to report items, but they don't know where the official lost-and-found channel is located. | **Invalidated** |
| **Assumption 2:** Social media and chat groups are sufficient for lost-and-found. | **Evidence:** Chat messages get buried rapidly under hundreds of general messages, making search impossible. | **Validated** |
| **Assumption 3:** Security offices keep an organized record of all items. | **Evidence:** Security offices use paper logbooks with vague descriptions, making item retrieval slow. | **Validated** |
| **Assumption 4:** Students will only use a platform if it has advanced IoT tags. | **Evidence:** Students only want a simple, searchable list with photo upload features; hardware tags are unnecessary. | **Invalidated** |
